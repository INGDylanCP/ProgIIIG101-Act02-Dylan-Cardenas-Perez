# ProgIIIG101-Act02-Dylan-Cardenas-Perez
# Backtracking y SLD - Prolog

## Descripción
Este proyecto modela un grafo de ciudades usando Prolog para consultar conexiones, costos de viaje y rutas con ciudades intermedias.

## Integrante
- Dylan Cardenas Perez

## Consultas principales
?- existeConexion(vancouver, edmonton).
?- conectadosDesdeyCosto(regina, Nodo, Costo).
?- tieneAristas(calgary).
?- viajeConCosto(vancouver, regina, Costo).
?- rutaConCosto(vancouver, winnipeg, Costo).

## Requisitos
SWI-Prolog

## Ejecución
Cargar el archivo y ejecutar las consultas en la consola de Prolog.
